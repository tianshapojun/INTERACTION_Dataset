# V.0.0

图(左)为去除轨迹优化模块后的模型控制，图(右)为添加后的模型控制。

当去除轨迹优化模块，无法根据导引线修正交通车前进路线，交通车有冲向对向车道的可能；

当添加轨迹优化模块，交叉口起止位置引导线稠密，阻碍路线判断。
